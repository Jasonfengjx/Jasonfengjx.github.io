python函数式编程

Created2023-10-07|Updated2023-10-07
|Post View:

通过对一整段代码分成几个函数,相当于给每一个独立的程序逻辑取名字,使得代码变成自解释的。

准则

函数式编程的准则：不依赖于外部的数据，而且也不改变外部数据的值，而是返回一个新的值给你。

特性

  • 不可变数据

  • first class functions

  • 尾递归优化

技术

map & reduce

  • 关注要干什么,而不是怎么干
  • 求正数的和(复杂的例子)

pipeline

只关心输入和输出

recursing

  • currying

higher order function

2）乘以3
3）转成字符串返回
